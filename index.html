Perguntas Frequentes

Tudo o que você precisa saber sobre o projeto.

1. O que é o GamingBR?
GamingBR é um launcher gratuito para PC que organiza e facilita o acesso a mais de 40.000 jogos de várias fontes. Você pode buscar, ver detalhes, instalar e gerenciar sua biblioteca de forma simples e rápida.
2. É realmente 100% gratuito?
Sim, completamente gratuito. Não tem paywall, assinatura, anúncios invasivos nem compras dentro do app. O foco é oferecer uma experiência boa e acessível para todos.
3. Como faço para baixar e instalar?
Basta clicar no botão "Baixar Agora" na página inicial. O instalador é compatível com Windows 10 e 11.
4. Preciso criar conta para usar?
Não é obrigatório, mas recomendado para salvar seu progresso, configurações de fontes e interagir com a comunidade através do Discord.
5. O que são "fontes" no GamingBR?
Fontes são os provedores de onde os metadados e links dos jogos são extraídos. Você tem controle total para ligar ou desligar cada uma delas no menu de configurações.
6. Como funciona a instalação dos jogos?
O GamingBR utiliza o motor aria2 para gerenciar downloads via torrent e links diretos, garantindo a velocidade máxima da sua internet.
7. O launcher é seguro?
Sim! Utilizamos protocolos de segurança modernos para autenticação e o código é focado em transparência. Milhares de usuários já validaram nossa ferramenta.
8. Por que alguns jogos não têm capa?
Estamos refinando nossa integração com a API IGDB. Se o jogo for muito obscuro, o sistema gera um card elegante automaticamente até que a capa seja encontrada.
9. Vai ter suporte para Emuladores?
Sim, está no nosso roadmap. O suporte ao RetroArch está em desenvolvimento avançado.
10. Como reportar um erro?
Nosso Discord é o hub principal de suporte. Entre lá, vá no canal de bugs e anexe um print. Nossa equipe responde rápido!
11. Requisitos para rodar o app?
Qualquer PC com Windows 10/11 64-bit e pelo menos 4GB de RAM roda o launcher tranquilamente.
12. Posso sugerir novos jogos?
Com certeza! Temos um canal de sugestões no Discord onde a comunidade vota no que deve ser priorizado.
13. Tem versão mobile?
Ainda não, mas o suporte a jogos Android e uma interface mobile estão nos planos de longo prazo.
14. O app salva na nuvem?
Suas configurações de biblioteca e perfil são sincronizadas na nossa database via Firebase.
15. O Windows acusou vírus, e agora?
Fica tranquilo, é o "Falso Positivo" do Windows SmartScreen por sermos desenvolvedores independentes. Clique em "Mais Informações" > "Executar assim mesmo".
